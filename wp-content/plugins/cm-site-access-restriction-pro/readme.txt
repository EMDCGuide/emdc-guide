== Description ==

A gated content powerful solution and restricted content plugin for WordPress. Support restricted content access by role on your WP site.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
